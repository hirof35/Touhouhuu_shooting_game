Touhou Java Engine - FinalJava Swingで構築された、本格的な弾幕シューティング（STG）ゲームエンジンです。東方Projectの伝統的なゲームシステムを継承し、軽量ながらも奥深い弾幕体験をブラウザレスなデスクトップ環境で提供します。🎮 ゲームシステム3段階の難易度: EASY、NORMAL、HARDから選択可能。難易度に応じて敵の弾速、発射頻度、弾幕の密度（Way数）がダイナミックに変化します。低速移動（精密操作）: Shiftキーを押している間は自機の移動速度が低下し、中心の当たり判定（赤点）が可視化されます。グレイズシステム: 敵弾の近くをかすめることで「GRAZE」が加算され、スコアが大幅に上昇します。スペルカード（ボム）: Xキーで画面内の敵弾を一掃し、一定時間の無敵状態を得ることができます。アイテム自動回収: 画面上部（境界線以上）に移動することで、画面内の全アイテムを自動的に引き寄せます。🕹 操作方法キーアクション方向キー自機の移動 / タイトルでの難易度選択Zキーショット / ゲーム開始 / リトライXキーボム（スペルカード）の発動Shiftキー低速移動（当たり判定表示）🛠 技術的なこだわりベクトル演算による弾幕: Math.atan2 や Math.hypot を駆使し、自機狙い弾や全方位弾の精密な軌道計算を行っています。ポリモーフィズムな管理: GameObject 抽象クラスを継承したリスト管理により、敵・弾・アイテムの更新処理を共通化し、メモリ効率を最適化しています。スムーズな描画: RenderingHints.KEY_ANTIALIASING による滑らかな描画と、16ms（約60FPS）のTimerによる快適な操作性を実現。🚀 クイックスタートJava環境（JDK 8以上）があれば、すぐにプレイ可能です。Bash# リポジトリのクローン
